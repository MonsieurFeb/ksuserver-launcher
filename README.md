# ksuserver-launcher
Специализированный лаунчер, выполняющий строго свою функцию - настройка доступа к КсюСерВеру.
В будущем может в него свои сборки закину.

Алгоритм кнопок (install+launch):
- устанавливает заданную версию игры
- устанавливает версию модлоадера (здесь могут быть ошибки, связанные с доступом к ресурсам, "проблема на стороне сервера")
- загружает архив со сборкой с гугл диска
- удаляет папку mods и разархивирует архив со сборкой
- добавляет айпи сервера в список
- запускает игру (авторизация Ely.by, подгрузка скина)

Поэтому нажимать данную кнопку стоит лишь только при первичной установке (обновлении) модпака.

Кнопка launch - делает лишь последнее действие

В планах - сделать авторизацию через OAuth, дизайн менюшки, отладить приложение (например при неправильно введённых данных, убрать консольку, нормально работающий буфер), возможно переписать на что-то более оптимизированное и прикрутить сертификат.
